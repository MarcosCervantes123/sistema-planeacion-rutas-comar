# sistema-planeacion-rutas-comar
Sistema en Java para registro de pedidos, asignación de rutas y unidades para Comar Logística y Transportes S.A. de C.V.
Descripcion del Proyecto

El Proyecto busca volver las rutas mas eficientes, reducir gastos y consumos, reducir tiempos de espera y reducir errores a la hora de cargar los pedidos de los clientes a las rutas.

Problematicas
- Rutas ineficientes
- Errores en pedidos
- Rutas con poca capacidad de carga
- Tiempos de entrega largos

Objetivo General
Desarrollar un sistema en JAVA que permita hacer lo siguiente:
- Dar de alta pedidos
- Asignar pedidos a una ruta
- Asignar una unidad a la ruta
- Generar salida
- Registrar entrega al cliente

Branches
- main: Version estable del proyecto
- develop: Rama para el desarrollo
